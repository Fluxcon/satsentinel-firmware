# SatSentinel Firmware

Pre-built firmware releases for SatSentinel ESP32-S3 devices.

## Latest Release

[Download latest firmware →](https://github.com/Fluxcon/satsentinel-firmware/releases/latest)

Or grab a specific version from the [releases page](https://github.com/Fluxcon/satsentinel-firmware/releases).

## Files

Each release folder contains:

| File                   | Description                          |
| ---------------------- | ------------------------------------ |
| `satsentinel.bin`      | App firmware — OTA upload via web UI |
| `bootloader.bin`       | Bootloader — first flash only        |
| `partition-table.bin`  | Partition table — first flash only   |
| `ota_data_initial.bin` | OTA data init — first flash only     |
| `version.txt`          | Build version string                 |
